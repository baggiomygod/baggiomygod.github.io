# import vs import * as

```import a from 'a.js'```
只导出```export default```

```import * as from 'a.js'```
导出所有

a.js
```
export const a = 1

export function fn () {
  console.log('this is fn...')
}

export default {
  x: 'x',
  y: 'y'
}

```

b.js
```
import a1 from './a'
import * as a2 from './a'

/*
    {x: 'x', y: 'y'}
*/
console.log(a1)

/*
    {
        a: 1,
        fn,
        default:{x: 'x', y: 'y'}
    }
*/
console.log(a2)
```
